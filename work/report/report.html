Explain the role of HTML, CSS and JavaScript in web development.
HTML acts as the structure where things are placed and how you can interact with the different parts. CSS acts as the appearance that decides how things should look. Every aspect of layout can be altered and controlled with css, thus making the task easier for a web designer. Javascript acts as the functionality when and how things should behave. Its not necesary to use javascript to make a webpage so its only there to enable interactivty.
